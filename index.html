  1. Enter same login and password as your intranet.

Article 002 – Remember Me feature

Steps:

  1. Close the application and run again.
  2. Enter same login and password as your intranet.
     
  3. Select “Remember Me” checkbox.
  4. Click “LOGIN” button. It should take you to the main home screen:
  5. Close the application and run again.
  6. It should take you to the main home screen without asking for credentials:

 

Article 003 – Use Menu

Steps:

  1. Once on the home page, click on the menu button in top left to dropdown the menu.


  2. Test all the menu items:
    1st Menu Item should show in the logged in person’s name.
    “Back” should take thre user to the previous screen.
    “Home” should take the user to the home screen.
    “Add Status” should take user to the screen to select equipment and add status.
    “View History” should take user to user’s past submissions.
    “Help” should take user to the help screen.
    “Sign Out” should sign user out and when user logs back in, should ask for credentials.

 

Article 004 –Search Equipment

Steps:

  1. Once on the home page, click on the “Add Status” button.
  2. The following screen should be shown:
  3. Enter partial or full Serial Number, Software Version, Revision or System Type in the search box. System should filter the equipment shown.
  4. Select a system type from the dropdown to narrow the search to specific system types. System should filter the equipment shown.

Article 005 –Scan QR Code or Barcode

Steps:

  1. On the equipment screen, click the barcode icon to the right of search box:
  2. App will use the rear camera to scan the QR code or barcode on the equipment and automatically extract the serial number for search.

Article 006 – Add Status and Save Draft

Steps:

  1. On the equipment screen, once the desired equipment is searched for, click the “Add Status” button for the equipment
  2. The add status screen is shown with the existing information already filled in.
  3. Fill in the Comments, “System condition on arrival” and “before leaving customer site” sections as required. Click the “Save Draft” Button.
  4. System should save the information without any validation and go to the user’s history screen.

Article 007 – Cancel Form

Steps:

  1. On the equipment screen, once the desired equipment is searched for, click the “Add Status” button for the equipment
  2. The add status screen is shown with the existing information already filled in.
  3. Click the “Cancel Form” Button. A confirmation box should appear. Click “Ok”. System should go to the home screen.

 

 

Article 008 –View History and Edit Comments

Steps:

  1. On the home screen, click the “View My History” button.


  2. User should be able to see all the past submissions made.


  3. User should be able to click on the plus sign to expand details.
  4. User should be able to write some comments and click add comments to add some comments to the submission history:

 

Article 009 –Edit History

Steps:

  1. On the “View My Hisotry” screen, click the red pencil besides each submission to edit the submission. Only draft submissions should be editable. All final submissions are read-only and red pencil icon is faded and disabled.
  2. The edit status screen is shown with the existing information already filled in.

 

Article 010 –Submit Status and receive email

Steps:

  1. On the equipment screen, once the desired equipment is searched for, click the “Add Status” button for the equipment
  2. The add status screen is shown with the existing information already filled in.
